# Stock Analysis Telegram Bot

Учебный проект: Telegram-бот для получения и анализа финансовых данных с использованием Alpha Vantage API.
Проект демонстрирует навыки работы с API, обработку данных и разработку Telegram-бота на Python.

---

## Возможности

- Получение текущей цены акции по тикеру.
- Анализ дневной динамики акции за выбранный период (5, 10, 30 дней).
- Получение текущего курса валютной пары (например, USD/EUR).
- Получение значения RSI (Relative Strength Index) для тикера и интерпретация его значения:
  - RSI ≤ 30 — возможная перепроданность (потенциальная покупка)
  - RSI ≥ 70 — возможная перекупленность (потенциальная продажа)
  - RSI между 30 и 70 — нейтральная зона
- Обработка ошибок и ограничений API.

Внимание: вся информация носит справочный характер и не является инвестиционной рекомендацией.
